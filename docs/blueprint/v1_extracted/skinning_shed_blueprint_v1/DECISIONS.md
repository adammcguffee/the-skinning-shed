# DECISIONS (Living Log)

## 2026-01-22
- MVP-first, production-ready launch philosophy (no throwaway v1).
- Platforms: Web + iOS + Android from day one (shared codebase).
- Premium modern UI/UX; outdoors “modern lodge” vibe; no neon/futuristic colors; not business/SaaS look.
- Trophy user page is called **Trophy Wall** (canonical term).
- BST tab is **The Swap Shop** (canonical term), discovery-only; no payments.
- Core species: Deer, Turkey, Bass.
- Additional tabs: Other Game, Other Fishing with comprehensive species dropdown + “Other (specify)” fallback.
- Weather tab with wind direction + historical attachments from harvest date/time & coarse location.
- Activity/Feeding Times calendar feature.
- Research/Patterns dashboard at State → County with aggregated condition buckets; privacy thresholds.
- Land for Lease / Land for Sale as first-class module with filters.
- GPS mapping/waypoints deferred (explicitly out of MVP).
